mahjong-scoring
================

RubyGem for calculating Mahjong scores (Chinese official rules).

Tile notation
-------------

```ruby
# circles
"1● 2● 3● 4● 5● 6● 7● 8● 9●"

# bamboo
"1/ 2/ 3/ 4/ 5/ 6/ 7/ 8/ 9/"

# characters
"1# 2# 3# 4# 5# 6# 7# 8# 9#"

# winds
"E W N S"
"東 西 北 南"

# dragons
"C F B"
"中 發 白"
```
